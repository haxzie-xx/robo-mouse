A simple cli node application to record your mouse clicks using iohook and replay 
it using robotjs.

Clone the repository
```
$ git clone https://github.com/haxzie/robo-mouse.git
```

Install dependencies
```
$ cd robo-mouse
$ npm install
```

Run the app
```
$ npm start
```
